# Operadroom

Autonomous execution layer for industrial digital twins.

**Operadroom** turns predictive asset alerts into completed maintenance workflows — bridging IoT telemetry, engineering knowledge, and enterprise ERP systems.

A subsidiary of [Softdroom Holdings](https://softdroom.com). Powered by [Reelin ID](https://reelin.id).

## Stack

- Next.js 15
- Tailwind CSS
- TypeScript

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy

Deploy to Vercel and connect `operadroom.com`.
